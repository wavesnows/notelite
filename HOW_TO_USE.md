## 🖱️ 操作步骤（图文说明）

### 场景1️⃣：置顶一个重要笔记

```
操作前的文件树：
├─ 📂 我的文件夹
│  ├─ 📄 笔记A
│  ├─ 📄 笔记B          ← 我想置顶这个
│  └─ 📄 笔记C

👉 操作：
1. 在左侧文件树找到"笔记B"
2. 用鼠标 **右键点击** "笔记B"
3. 会弹出一个菜单，选择 "📍 Pin Note"

操作后的文件树：
├─ 📂 我的文件夹
│  ├─ 📄 笔记B 📌       ← 自动移到顶部，带有📌标签
│  ├─ 📄 笔记A
│  └─ 📄 笔记C
```

---

### 场景2️⃣：给笔记添加星标收藏

```
操作前：
📄 重要文档

👉 操作：
1. 右键点击 "重要文档"
2. 在菜单中选择 "☆ Add Star"

操作后：
📄 重要文档 ⭐          ← 出现星星图标
```

---

### 场景3️⃣：取消置顶

```
当前状态：
📄 笔记B 📌            ← 已置顶的笔记

👉 操作：
1. 右键点击 "笔记B"
2. 菜单会显示 "📌 Unpin Note"（注意：文字变了！）
3. 点击它

结果：
📄 笔记B               ← 📌标签消失，笔记回到原位置
```

---

### 场景4️⃣：取消星标

```
当前状态：
📄 重要文档 ⭐

👉 操作：
1. 右键点击 "重要文档"
2. 菜单会显示 "⭐ Remove Star"（注意：文字变了！）
3. 点击它

结果：
📄 重要文档            ← ⭐图标消失
```

## 📋 完整的右键菜单说明

### 当你右键点击 **笔记文件** 时，会看到：

```
┌─────────────────────────┐
│ 📍 Pin Note            │ ← 置顶笔记（如果未置顶）
│ 📌 Unpin Note          │ ← 取消置顶（如果已置顶）
│ ☆ Add Star             │ ← 添加星标（如果未收藏）
│ ⭐ Remove Star         │ ← 取消星标（如果已收藏）
│ ❌ Remove              │ ← 删除笔记
└─────────────────────────┘
```

**注意**：菜单项会根据当前状态动态变化！
- 如果笔记已置顶，显示"Unpin"
- 如果笔记未置顶，显示"Pin"
- 星标同理

### 当你右键点击 **文件夹** 时，会看到：

```
┌─────────────────────────┐
│ 📄 File                │ ← 创建新笔记
│ 📂 Folder              │ ← 创建子文件夹
│ ❌ Remove              │ ← 删除文件夹
└─────────────────────────┘
```

## 🔍 如何知道功能是否正常工作？

### ✅ 置顶成功的标志：
1. 笔记名称后面出现 **📌** 标签
2. 笔记自动移动到文件夹顶部
3. 再次右键点击，菜单显示"📌 Unpin Note"

### ✅ 收藏成功的标志：
1. 笔记名称后面出现 **⭐** 图标
2. 再次右键点击，菜单显示"⭐ Remove Star"

### ✅ 持久化成功的标志：
1. 关闭应用
2. 重新运行 `npm run dev`
3. 打开后，之前置顶和收藏的笔记状态依然保留

## 💡 使用技巧

### 技巧1：快速访问常用笔记
将每天要用的笔记置顶，这样打开应用就能立即看到它们在顶部。

### 技巧2：标记重要笔记
用星标标记重要但不需要每天看的笔记，方便以后识别。

### 技巧3：多个置顶笔记
可以置顶多个笔记，它们会按照置顶的顺序显示在顶部。

### 技巧4：嵌套文件夹
在子文件夹中的笔记也可以置顶，置顶效果只在当前文件夹内生效。

## ❓ 常见问题

### Q1: 我右键点击了，但没看到"Pin Note"选项？
**A**: 请确保：
- 你点击的是 **笔记文件**（不是文件夹）
- 应用已经重新编译（运行了 `npm run dev`）
- 检查控制台是否有错误

### Q2: 点击了"Pin Note"，但笔记没有移到顶部？
**A**:
- 刷新一下文件树（切换到其他笔记本再切回来）
- 检查浏览器控制台是否有错误信息
- 确保笔记在同一个文件夹内

### Q3: 关闭应用后，置顶和收藏都没了？
**A**:
- 检查 electron-store 是否正常工作
- 查看控制台是否有存储错误
- 确保应用有写入权限

### Q4: 可以同时置顶和收藏吗？
**A**: 可以！一个笔记可以同时被置顶和收藏，会同时显示 ⭐ 和 📌。

### Q5: 删除笔记后，它还在置顶列表里吗？
**A**: 不会。删除笔记时会自动从置顶和收藏列表中移除。

## 🐛 调试方法

如果遇到问题，可以打开开发者工具查看：

1. 应用运行时，按 `Cmd+Option+I`（Mac）或 `Ctrl+Shift+I`（Windows）
2. 切换到 Console 标签
3. 查看是否有红色错误信息
4. 可以在 Console 中输入以下命令查看当前状态：

```javascript
// 查看置顶列表
console.log(ttsStore.favorites.pinned)

// 查看收藏列表
console.log(ttsStore.favorites.starred)
```
