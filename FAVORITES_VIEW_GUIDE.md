# ⭐ 收藏夹视图使用指南

## 🎉 新功能：收藏夹视图

现在可以在侧边栏查看所有星标笔记的集中视图了！

---

## 📋 如何使用

### 第1步：打开收藏夹视图

在左侧最窄的侧边栏，你会看到两个图标：

```
┌─────┐
│ 📓  │ ← 笔记本（文件树视图）
│ ⭐  │ ← 收藏夹（新增！）
│     │
│ ⚙️  │ ← 设置（在底部）
└─────┘
```

**点击 ⭐ 图标**，就会切换到收藏夹视图。

---

### 第2步：查看收藏的笔记

收藏夹视图会显示：

```
收藏夹                           3
─────────────────────────────────

⭐ 重要文档 📌                    ✕
⭐ 项目计划                       ✕
⭐ 会议记录                       ✕
```

**显示内容：**
- 📊 **收藏数量** - 右上角显示总共有多少个收藏
- ⭐ **星标图标** - 每个笔记前都有星标
- 📌 **置顶标记** - 如果笔记同时被置顶，会显示 📌
- ✕ **移除按钮** - 鼠标悬停时显示，点击可取消收藏

---

### 第3步：打开笔记

**点击任意收藏的笔记**，就会在编辑器中打开它。

---

### 第4步：管理收藏

#### 添加到收藏：
1. 切换回文件树视图（点击 📓 图标）
2. 右键点击要收藏的笔记
3. 选择 "☆ 添加星标"
4. 切换到收藏夹视图，就能看到新添加的笔记

#### 取消收藏：
**方法1：在收藏夹视图**
- 鼠标悬停在笔记上
- 点击右侧的 ✕ 按钮

**方法2：在文件树视图**
- 右键点击已收藏的笔记
- 选择 "⭐ 移除星标"

## ✨ 功能特点

### 1. 集中管理
所有星标笔记集中在一个地方，不用在文件树中寻找。

### 2. 快速访问
点击即可打开笔记，比在文件树中导航更快。

### 3. 实时更新
- 在文件树中添加星标，收藏夹立即显示
- 在收藏夹中移除星标，立即从列表消失
- 显示实时的收藏数量

### 4. 显示置顶状态
如果笔记同时被置顶，会显示 📌 图标。

### 5. 悬停显示操作
鼠标悬停在笔记上时，才显示移除按钮，界面更简洁。

## 🔄 视图切换

### 在不同视图间切换：

```
📓 文件树视图
↓ 点击 ⭐
⭐ 收藏夹视图
↓ 点击 📓
📓 文件树视图
```

**提示：** 活动的图标会有高亮显示（左侧有竖线）。

---

## 🎯 与置顶功能的配合

### 最佳实践：

```
📌 置顶 + ⭐ 星标 = 超级重要
📌 只置顶 = 临时重要，常用
⭐ 只星标 = 长期重要，不常用
```

**示例：**
- **项目文档** → 📌⭐（既置顶又收藏）
- **今日待办** → 📌（只置顶，处理完就取消）
- **参考资料** → ⭐（只收藏，需要时查看）

## 🐛 故障排除

### 问题1：看不到 ⭐ 图标

**解决：**
```bash
# 刷新应用
按 Cmd + R
```

---

### 问题2：收藏夹是空的，但我添加过星标

**检查：**
1. 切换到文件树视图
2. 查看笔记旁边是否有 ⭐ 图标
3. 如果有，切换回收藏夹应该能看到
4. 如果没有，重新添加星标

---

### 问题3：点击笔记没反应

**解决：**
1. 检查笔记文件是否存在
2. 查看控制台是否有错误
3. 尝试在文件树中打开该笔记

---

## 📊 数据同步

### 收藏夹与文件树完全同步：

```
文件树中添加星标 → 收藏夹立即显示
文件树中移除星标 → 收藏夹立即移除
收藏夹中移除星标 → 文件树中星标消失
删除笔记文件 → 自动从收藏夹移除
```
