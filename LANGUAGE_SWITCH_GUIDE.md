# 🌐 语言切换功能使用指南

## 功能说明

现在可以在设置面板中切换软件的显示语言（中文/英文）。

---

## 📋 如何使用

### 第1步：打开设置面板

1. 在应用左下角，找到三个圆形按钮
2. 点击最右边的 **⚙️ 设置** 按钮（齿轮图标）
3. 设置面板会从右侧滑出

---

### 第2步：切换语言

1. 在设置面板的 **"Global Setting"** 标签页
2. 找到第一个选项：**"Display Language"** 或 **"显示语言"**
3. 点击下拉菜单，选择：
   - **English** - 英文界面
   - **Chinese** - 中文界面
4. 选择后会立即切换，并显示成功提示

---

### 第3步：保存设置

语言设置会**自动保存**，下次打开应用时会记住你的选择。

---

## 🛠️ 技术实现

### 文件修改：

1. **`/src/i18n/en.ts`** - 添加英文翻译
2. **`/src/i18n/zh.ts`** - 添加中文翻译
3. **`/src/i18n/index.ts`** - 从 store 读取保存的语言
4. **`/src/store/store.ts`** - 添加语言状态和保存方法
5. **`/src/components/aside/AConfig.vue`** - 添加语言切换 UI

### 数据存储：

- **存储位置**：electron-store
- **存储键**：`language`
- **存储值**：`'en_US'` 或 `'zh_CN'`

## 🐛 故障排除

### 问题1：看不到语言选项

**解决方法：**
```bash
# 刷新应用
按 Cmd + R

# 或重启应用

```

---

### 问题2：切换后没有变化

**原因：** 该部分内容还没有添加翻译

**解决方法：**
只有已经使用 `t('...')` 函数包裹的文字才会翻译。目前已翻译的内容有限，后续会逐步完善。

---

### 问题3：切换后部分是英文，部分是中文

**这是正常的！**

目前只有部分功能添加了多语言支持：
- ✅ 搜索功能
- ✅ 收藏功能
- ✅ 语言设置

其他部分（如设置面板的其他选项）会在后续版本中逐步翻译。

## 📚 如何添加更多翻译

如果你想翻译更多内容：

### 1. 在翻译文件中添加键值对

**en.ts:**
```typescript
settings: {
  localPath: 'Local Path',
  changePath: 'Change Default Path',
}
```

**zh.ts:**
```typescript
settings: {
  localPath: '本地路径',
  changePath: '更改默认路径',
}
```

### 2. 在 Vue 组件中使用

```vue
<template>
  <el-form-item :label="t('settings.localPath')">
    <el-button>{{ t('settings.changePath') }}</el-button>
  </el-form-item>
</template>

<script setup>
import { useI18n } from 'vue-i18n';
const { t } = useI18n();
</script>
```
