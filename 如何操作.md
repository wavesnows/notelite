## 🎯 第二步：找到菜单入口

应用启动后，看左侧的文件树，你会看到：

```
📂 我的文件夹 ⚙︎
  📄 笔记A ⚙︎          👈 看到这个齿轮图标了吗？
  📄 笔记B ⚙︎
  📄 笔记C ⚙︎
```

**每个笔记和文件夹右边都有一个 ⚙︎ 图标！**

---

## 🖱️ 第三步：打开菜单

### 操作方法：

```
📄 我的笔记 ⚙︎
         ↑
    点击这里！（鼠标左键单击）
```

**不是右键！是左键点击 ⚙︎ 图标！**

---

## 📋 第四步：选择功能

点击 ⚙︎ 后，会弹出菜单：

### 如果是笔记文件，菜单显示：
```
┌─────────────────────────┐
│ 📍 Pin Note            │ ← 点这个可以置顶
│ ☆ Add Star             │ ← 点这个可以收藏
│ ❌ Remove              │ ← 点这个可以删除
└─────────────────────────┘
```

### 如果是文件夹，菜单显示：
```
┌─────────────────────────┐
│ 📄 File                │ ← 创建新笔记
│ 📂 Folder              │ ← 创建子文件夹
│ ❌ Remove              │ ← 删除文件夹
└─────────────────────────┘
```

## 🐛 如果还是看不到菜单

### 方法1：完全重启应用

```bash
# 在终端中：
# 1. 按 Ctrl+C 停止应用
# 2. 等待完全停止
# 3. 重新运行
npm run dev
```

---

### 方法2：清理缓存重新编译

```bash
# 停止应用（Ctrl+C）
# 清理编译缓存
rm -rf dist/
# 重新启动
npm run dev
```

---

### 方法3：检查错误

打开开发者工具查看错误：

1. 在应用窗口按 `Cmd + Option + I`（Mac）
2. 切换到 "Console" 标签
3. 看是否有红色错误信息
4. 把错误信息告诉我

## 📝 操作要点总结

### ✅ 正确操作：
1. **重启应用**（重要！）
2. 找到笔记旁边的 **⚙︎** 图标
3. **左键点击** ⚙︎ 图标
4. 在弹出的菜单中选择功能

### ❌ 常见错误：
- ❌ 没有重启应用
- ❌ 找不到 ⚙︎ 图标
- ❌ 用右键点击笔记名称（应该点击⚙︎）
- ❌ 点击了文件夹的⚙︎（文件夹菜单不一样）
