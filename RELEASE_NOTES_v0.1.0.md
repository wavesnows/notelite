# YesnoteLite v0.1.0 Release Notes

## 🎉 新版本发布

这是 YesnoteLite 的第一个重大更新版本，包含 8 个重要改进和功能增强。

---

## ✨ 新功能

### 1. 保存状态指示器
- 实时显示保存状态（保存中/已保存/保存失败）
- 位于窗口底部状态栏右侧
- 3秒后自动消失，不干扰工作

### 2. 自动保存
- 每30秒自动保存未保存的更改
- 智能检测内容变化
- 防止意外关闭导致数据丢失

### 3. 快捷键帮助
- 按 `Cmd+?` (Mac) 或 `Ctrl+?` (Windows/Linux) 打开帮助
- 显示所有可用快捷键
- 支持中英文界面

### 4. 面包屑导航
- 编辑器顶部显示完整文件路径
- 清晰展示当前笔记位置
- 便于理解文件组织结构

---

## 🚀 性能优化

### 5. 文件扫描优化
- 使用防抖机制减少不必要的文件扫描
- 连续操作只触发一次刷新
- 大量笔记时响应更快

### 6. 响应式编辑器高度
- 编辑器高度自动适应窗口大小
- 不同屏幕尺寸都有最佳体验
- 移除固定高度限制

---

## 🎨 界面改进

### 7. 统一按钮样式
- 移除不规范的 CSS transform
- 统一按钮尺寸和间距
- 更加专业和美观的界面

### 8. 统一错误处理
- 标准化的错误提示
- 中英文错误消息
- 更友好的用户反馈

---

## 🐛 Bug 修复

- 修复置顶文件后树菜单展开状态丢失问题
- 修复文件树图标风格不一致问题
- 修复过滤输入框样式突兀问题
- 修复编辑器宽度不响应窗口大小问题

---

## 📋 快捷键列表

### 通用
- `Cmd/Ctrl + ?` - 显示快捷键帮助
- `Cmd/Ctrl + S` - 保存笔记

### Git 操作
- `Cmd/Ctrl + Shift + P` - 推送到远程

### 编辑器
- `Cmd/Ctrl + Shift + H` - 插入标题
- `Cmd/Ctrl + Shift + W` - 插入警告框
- `Cmd/Ctrl + B` - 粗体
- `Cmd/Ctrl + I` - 斜体
- `Cmd/Ctrl + K` - 插入链接

---

## 💡 使用提示

1. **自动保存**: 编辑后无需手动保存，30秒后会自动保存
2. **保存状态**: 查看右下角状态栏了解保存状态
3. **快捷键**: 按 `Cmd+?` 随时查看所有快捷键
4. **面包屑**: 编辑器顶部显示当前文件路径
5. **置顶**: 右键笔记选择"置顶"，文件会自动移到顶部

## 📦 安装说明

### macOS
1. 下载 `YesnoteLite-0.1.0-x64.dmg` (Intel) 或 `YesnoteLite-0.1.0-arm64.dmg` (Apple Silicon)
2. 打开 DMG 文件
3. 将 YesnoteLite 拖到 Applications 文件夹
4. 首次打开可能需要在"系统偏好设置 > 安全性与隐私"中允许

### Windows
1. 下载 `YesnoteLite-0.1.0-Setup.exe`
2. 运行安装程序
3. 按照提示完成安装

### Linux
1. 下载 `YesnoteLite-0.1.0.AppImage`
2. 添加执行权限: `chmod +x YesnoteLite-0.1.0.AppImage`
3. 运行: `./YesnoteLite-0.1.0.AppImage`
